Steps in creating authentication

1.) create a model email must be unique

2.) create a function for login and sign up(hash password)

3.) perform checks for invalid email and passsword

4.) generate web token

5.) create a global context state for users

6.) crate pages and functions for signing up, logging  logging out

7.) Protect the routes from logged out users.         

8.) make ath request from front end and set auth headers

9.) protect frontend pages from logged out users

10.) make sure each user sees their own personal details(don't forget to create model for user id)
